# Personal Portfolio Website

This repository contains the source code for my personal portfolio website. It is designed to provide a clear, modern, and fast digital presence that highlights my work, skills, experience, and projects.

## Overview

The portfolio serves as my professional identity on the web. It features a clean layout, modular design, and fast performance enabled by a modern development stack. The project setup is minimal and easy to maintain, allowing smooth customization and future expansion.

## Features

- Modern and responsive user interface
- Component-based architecture
- Fast development and build tooling
- Clean folder structure
- Easy deployment to multiple hosting platforms
- Separation of static assets and source code

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
- Node.js

## Project Structure

- my-portfolio/
- ├── public/               # Static assets

- │   ├── components/

- │   └── styles/
- ├── package.json
- ├── vite.config.js
- └── README.md

## Installation and Setup

### Prerequisites
- Node.js (LTS recommended)
- npm or yarn

### Clone the Repository
```
git clone https://github.com/MPrajwalKini/my-portfolio.git
cd my-portfolio
```

### Install Dependencies
```
npm install
```
or
```
yarn install
```

### Start the Development Server
```
npm run dev
```

### Build for Production
```
npm run build
```

The optimized output will be generated inside the dist/ directory.

## Deployment

You can deploy this project on any modern static hosting service, including:

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

You may either upload the built dist/ directory or connect the repository for continuous deployment.
